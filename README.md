# Rock Paper Scissors Game:

This project is a simple Rock Paper Scissors game built using Python.  
It is a menu-driven program where the user plays against the computer.

---

## Features:

- Play Rock, Paper, Scissors
- Computer generates random choice
- Game logic using functions
- Menu-driven system using while loop
- Displays result after each round

---

## Technologies Used:

- Python (Basic Concepts)
- Functions
- Dictionary
- Random Module
- Conditional Statements
- Loop (while True)

---

## Project Structure:

rock_paper_scissors/
│
├── main.py         # Menu and user interaction
└── game_logic.py   # Game logic and result calculation

---

## How to Run:

1. Open terminal in project folder  
2. Run:
python main.py

---

## How the Game Works:

- User selects:
  - r → Rock  
  - p → Paper  
  - s → Scissor  
- Computer randomly selects one option  
- Winner is decided based on game rules
